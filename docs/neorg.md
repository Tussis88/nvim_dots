# Neorg

## Main Keybindings for [Neorg](https://github.com/nvim-neorg/neorg)


|key                   |description                    |
| :---------------: | :----------------: |
|        `,`        |   local leader    |
|   `<leader>ww`    | open the index of neorg |
| `<localleader>nn` |     create a new note     |
|       `za`        | fold current header |
